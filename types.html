More often than not one can find that computers come with a pre-loaded operating system.The operating system is not set in stone, more often than not users change to one of the following: Microsoft Windows, Mac OS X, and Linux.


Microsoft Windows was created in the mid-1980's and has been modified throgh the course of the years. The current version is Windows 10 which was released in 2015. Windows is often pre-loaded on most PC's.




Mac OS X is an operating system that was created by the well known brand of cell phones, none other than Apple. This software can be found in Macintosh computers(Macs). The latest software is OS X with the latest version being that of El Capitan which was released in 2015.



Linux is an open-source operating system, that can be modified and distributed by anyone around the world. Seeing as it is not a proprietary software like the ones mentioned above, this allows for it modification by anyone not just one particular company.
